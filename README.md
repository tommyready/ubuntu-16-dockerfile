## Build Steps

[Docker Download](https://www.docker.com/products/docker-desktop)

```bash
    git clone git@github.com:tommyready/ubuntu-16-dockerfile.git
    cd ubuntu-16-dockerfile
    docker build . 
``
*don't forget the . after the build command
